# Banking-Management-System

    Version Française

Projet académique Java (POO) : Système de gestion bancaire en console développé à l'Université de Yaoundé I. 🏦💻

    📌 Présentation du projet

        Ce projet consiste en l'implémentation complète d'une application de gestion bancaire en mode console, développée en Java. Il a été réalisé en équipe dans le cadre de notre formation en Licence 1 Informatique à l'Université de Yaoundé I pour l'année académique 2025-2026, sous la direction du Dr KIMBI.
        L'objectif principal est de mettre en pratique les concepts fondamentaux de la Programmation Orientée Objet (POO) — à savoir l'encapsulation, l'héritage et le polymorphisme — tout en exploitant les structures de données dynamiques en mémoire (ArrayList).

    ✨ Fonctionnalités clés

    -Gestion complète des comptes : Création, modification et suppression de comptes avec génération automatique d'un numéro unique à 9 chiffres.
    -Types de comptes gérés : Prise en charge des Comptes Courants, Épargne et Professionnels, chacun appliquant des frais de gestion mensuels distincts.
    -Opérations financières courantes : Gestion des dépôts, des retraits et des virements de compte à compte avec authentification par mot de passe.
    -Suivi et Historique : Consultation en temps réel du solde et suivi chronologique de toutes les transactions effectuées (y compris la création du compte).
    -Sécurité & Robustesse : Validation des entrées utilisateur (comme la vérification de l'âge minimum de 18 ans pour l'ouverture d'un compte) et gestion propre des erreurs de saisie pour éviter les plantages.

    🛠️ Technologies et outils

    -Langage : Java (Version 8 ou supérieure)
    -Bibliothèques standards : java.util.ArrayList, java.util.Scanner et l'API de gestion du temps (LocalDate, Period, DateTimeFormatter)
    -Environnements recommandés : Eclipse, IntelliJ IDEA ou NetBeans


      English Version
      
Academic Java (OOP) Project: A console-based Core Banking Management System developed at the University of Yaoundé I. 🏦💻

        📌 Project Overview

    This project is a fully functional, console-based Banking Management System built from scratch using Java. It was developed as a collaborative group project during our BSc in Computer Science (Licence 1 Informatique) at the University of Yaoundé I (Academic Year 2025-2026), under the supervision of Dr. KIMBI.
    The primary focus of this assignment is to implement key Object-Oriented Programming (OOP) concepts, such as encapsulation, inheritance, and polymorphism, while utilizing Java dynamic memory collections (ArrayList).

        ✨ Key Features

    -Account Lifecycle Management: Support for opening, modifying, and closing banking accounts, featuring an automatic unique 9-digit account number generator.
    -Multiple Account Types: Tailored management for Checking (Courant), Savings (Épargne), and Business (Professionnel) accounts, each with its own monthly fee structure.
    -Financial Transactions: Secure handling of cash deposits, withdrawals, and account-to-account transfers, fully protected by password authentication.
    -Audit Trail & Tracking: Real-time balance lookup and a complete chronological logging system tracking all account operations from day one.
    -Reliability & Validation: Built-in validation algorithms (such as checking that applicants are 18 or older) and robust user-input error handling to ensure application stability.

        🛠️ Tech Stack & Tools

    -Language: Java (Version 8 or higher)
    -Core Libraries: java.util.ArrayList, java.util.Scanner, and the java.time core API (LocalDate, Period, DateTimeFormatter)
    -Recommended IDEs: Eclipse, IntelliJ IDEA, or NetBeans.
